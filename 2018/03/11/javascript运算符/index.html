javascript运算符制作计算器

发表于 2018-03-11 | 分类于 javascript

计算器

  • 今天看了老师讲解的一个计算器用js制作的方法，用的是最基础的JS代码写的，逻辑思维很强，课后，我自己也写了一遍，值得一观，我把这个分享出来，随便说这个大致的思路：
  • 计算器上的数字，我们每一次按都会有一个value值，这个值用一变量存起来，比如按下的是9然后我们又按+号然后再按9，我们要分别把拿9和+和9，所以要用三个变量分别把这个值存起来，这时候要用到判断，如果存符号的变量里面有符号了比如存了个+号，这时候就要把+号后面的9给存起来，如果没有符号就要把前面的9存起来。然后封装个函数进行运算，把运算结果也存起来，这样就可以接着实现连续运算，把结果再赋给存第一个值的变量，因为，只要有第三个要参加运算，就先把前两个运算出来，拿到这个值再运算第三个！这样就差不多已经实现连续运算了。
